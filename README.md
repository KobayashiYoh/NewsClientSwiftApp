# NewsClientSwiftApp
<img src="https://user-images.githubusercontent.com/82624334/209466801-523b7fae-6863-4769-8320-d485109f68ac.png" width="300">

## 使用した技術
Swift, SwiftUI

## 機能
* News APIからニュース記事を取得
* 取得したニュース記事を表示
  * 見出し
  * 画像
  * 投稿日時
  * 投稿者
* 検索機能

## 取り組んだきっかけ
Flutterだけでなく、ネイティブでのアプリ開発もできるようになりたいと思い、Swiftのアウトプットを兼ねて開発を開始。
